# About the Project

Project developed for the Kobe - Creative Software House.

The following resources were charged for this project:

• Scroll through a list of upcoming movies - including the movie name, poster, or background.
image, genre and release data. A list should not limit only the first 20 movies
returned by the API.

• Select a specific movie to display details (name, poster image, genre, overview, and version)
meeting).

• Search for movies by entering a partial or full movie name (optional).

## About the App

I chose to use Angular because of the data consumption via API and because there was no need to store data.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.2.2.

## About the dependencies

• ngx-infinite-scroll => Used to add infinite scroll function.
"npm install ngx-infinite-scroll --save"

• ngx-spinner => Used to add load points in infinite scroll.
"npm install ngx-spinner --save"

• ngx-animate => Used to add animation effects.
"npm install ngx-animate --save"

• ngx-wow =>  Used to control animations, depends on the "ngx-animate"
"npm install --save wowjs"
"npm install --save ngx-wow"

## Development server

Add the "node_modules" folder to the project directory.

Run `ng serve -o` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
